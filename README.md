# CalmingWebpage
一个响应式的白噪音网页😮

用于测试或填充空白网站服务器

## 本地测试方法
Windows下推荐 [simple-http-server](https://github.com/jeske/SimpleHttpServer)
把它放到根目录，然后新建脚本文件并运行
```
simple-http-server.exe -i -p 80
pause
```
Linux下就各显神通吧😂
我是懒人我用docker跑nginx
